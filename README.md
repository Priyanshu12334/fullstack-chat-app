# ChatCrypt - Full Stack Realtime Chat Application

ChatCrypt is a modern full-stack realtime chat application built with the MERN stack. It provides secure authentication, instant messaging, online user presence, and a responsive user experience powered by modern web technologies.

## 🚀 Features

* 🔐 Secure Authentication & Authorization using JWT
* 👤 Unique Username System for user discovery
* 🔍 Search Users by Username and Start Conversations
* 🔑 Login using Email or Username
* ✏️ Editable User Profile (Name, Username, Bio & Profile Picture)
* 💬 Realtime Messaging with Socket.io
* 🟢 Online User Status Tracking
* 🎨 Modern UI built with Tailwind CSS
* ⚡ Fast and Responsive User Experience
* 🗄️ MongoDB Database Integration
* 🔒 Protected Routes and Persistent User Sessions

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Socket.io Client

### Backend

* Node.js
* Express.js
* MongoDB
* Socket.io
* JWT Authentication
* Cloudinary

## 📦 Environment Variables

Create a `.env` file in the root directory and add the following variables:

```env
MONGODB_URI=your_mongodb_connection_string

PORT=5001

JWT_SECRET=your_jwt_secret

CLOUDINARY_CLOUD_NAME=your_cloudinary_cloud_name
CLOUDINARY_API_KEY=your_cloudinary_api_key
CLOUDINARY_API_SECRET=your_cloudinary_api_secret

NODE_ENV=development
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/chatcrypt.git
cd chatcrypt
```

## 🌐 Live Demo

- **Live Application:** https://fullstack-chat-app-seven-tawny.vercel.app

Install dependencies:

```bash
npm install
```

---

## 🏗️ Build for Production

```bash
npm run build
```

---

## ▶️ Run the Application

Start the production server:

```bash
npm start
```

For development:

```bash
npm run dev
```
